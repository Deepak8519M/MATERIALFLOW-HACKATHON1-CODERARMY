📘 Day 2: HTML Essentials - Structure Your First Webpage




Welcome to Day 2 of MaterialFlow's lecture series, where we dive deep into HTML essentials! Learn how to structure web content using core tags like <h1> to <h6>, <p>, <ul>, <ol>, <a>, and <img>. This beginner-friendly guide walks you through setting up a VS Code environment, applying semantic tags, and building a mini-project to create a functional webpage from scratch. 🚀


Progress: 35% (Day 2 of 06)

1. Understanding HTML: HyperText Markup Language 🌐

HTML Overview
HTML stands for HyperText Markup Language, the backbone of web development. It enables the creation of structured web pages with hyperlinks for navigation and tags for semantic meaning. The term "HyperText" refers to links connecting pages, as seen in the first website (World Wide Web). "Markup Language" involves using tags to define content structure, like headings and paragraphs, making content meaningful for browsers and search engines.

Key Concepts
  • HyperText: Links pages for seamless navigation.
  • Markup: Tags like <h1> and <p> give content structure.
  • Purpose: Builds the skeleton of a webpage, defining its layout and meaning.

3. HTML Tags and Elements: Building Blocks of Webpages 📝

3.1. Why Tags Matter

Without tags, text in HTML appears unstructured and flat. Tags like <h1> and <p> add semantic meaning and structure, making content readable for users and machines. For example, wrapping "Web Development Course" in <h1> marks it as the main heading, while <p> defines paragraphs. 


<h1>Web Development Course</h1>
<p>In this course, learn to become a full stack developer from zero to advanced.</p>

3.2. Heading Tags (h1-h6)



HTML provides six heading tags: <h1> to <h6>. They define content hierarchy, with <h1> being the most important (used once per page) and <h6> the least.

Tag Purpose Example
<h1> Main page title <h1>Welcome to HTML</h1>
<h2> Major section heading <h2>Introduction</h2>
<h3> Sub-section heading <h3>Basics of Tags</h3>

SEO Importance: Headings help search engines understand content hierarchy. Use one <h1> per page and maintain a logical sequence (e.g., <h1> → <h2> → <h3>).

3.3. Paragraph Tags



The <p> tag defines paragraphs, separating blocks of text with spacing. Use it for regular content, not headings. 


<p>This course covers HTML basics for beginners.</p>
<p>Explore tags, attributes, and more!</p>

Tip: Use lorem50 in VS Code to generate 50 words of placeholder text for testing.

3.4. Horizontal Rule and Line Break Tags



Horizontal Rule (<hr>): A self-closing tag that draws a thematic break (line) to separate content.
Line Break (<br>): A self-closing tag that inserts a single line break within text. 


<p>Topic 1</p>
<hr />
<p>Topic 2</p>
<p>Address: 221B Baker Street<br />London, England</p>

3.5. List Tags (Ordered and Unordered)



Lists organize related items. Use <ul> for unordered (bulleted) lists and <ol> for ordered (numbered) lists, with <li> for each item.

List Type Tag Example
Unordered <ul> 

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>
Ordered <ol> 

<ol>
  <li>Boil water</li>
  <li>Add tea</li>
</ol>

3.6. Anchor Tags and Attributes

The <a> tag creates hyperlinks. Use the href attribute for the destination URL and target="_blank" to open links in a new tab. Attributes add properties to tags. 


<!-- Example Anchor Tag -->
<a href="https://coderarmy.in" target="_blank">Visit Coder Army</a>

Comments: Use <!-- Comment --> to add notes invisible to users.

3.7. Image Tags

The <img> tag embeds images. It’s self-closing and requires src (image path) and alt (accessibility text). Use height and width for sizing. 


<img src="https://cdn.britannica.com/16/234216-050-C66F8665/beagle-hound-dog.jpg" height="200px" width="200px" alt="Beagle Dog">

5. Key Summary 📌

What is HTML?
HTML (HyperText Markup Language) is the foundation of web pages, using tags to structure content and enable navigation via hyperlinks.

Key Tags and Their Uses
  • Headings (<h1>-<h6>): Define content hierarchy, with <h1> as the main title.
  • Paragraph (<p>): Structures regular text content.
  • Anchor (<a>): Creates hyperlinks with href and target attributes.
  • Image (<img>): Embeds images with src and alt attributes.
  • Lists (<ul>, <ol>, <li>): Organize items as bulleted or numbered lists.
  • Horizontal Rule (<hr>): Adds thematic breaks.
  • Line Break (<br>): Inserts line breaks within text.
